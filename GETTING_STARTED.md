## ðŸš€ Quick Start (3 minutes)

```bash
# 1. Install system dependencies (Ubuntu/Debian)
sudo apt install python3.11 python3.11-venv mysql-server

# 2. Run quick start script
cd careersadhana
chmod +x quick_start.sh
./quick_start.sh

# 3. Setup database
mysql -u root -p < manual_create_db.sql
# (Edit manual_create_db.sql first to change password!)

# 4. Edit .env file
nano .env
# Add your database password and other settings

# 5. Migrate database
source venv/bin/activate
python manage.py migrate

# 6. Create admin user
python manage.py createsuperuser

# 7. Run development server
python manage.py runserver
```

Visit: **http://localhost:8000**

Admin: **http://localhost:8000/admin**
